# AtmosViz 要件 (2025-10-19 日本語版)

英語版: [requirements.md](requirements.md)

## 機能要件
| ID | 説明 | 状況 |
|----|------|------|
| FR-01 | Inside / Outside の室内可視化を用意し、Home, Front, Back, Left, Right, Top, User の各カメラプリセットを選択可能にする。 | 完了 |
| FR-02 | Inside のズーム下限を 40%以上（Top は 25%以上）に制限し、既定値を 60%（Top は 25%）とする。 | 完了 |
| FR-03 | 密度コントロール（レベル 1〜5）と色凡例を備えた Radiation Heatmap ビューを提供する。 | 完了 |
| FR-04 | Temporal Trails と Peak Hold 可視化を同一カメラ制御で提供する。 | 完了 |
| FR-05 | Low / Mid / High の色重み付けをスライダーと Colour Mix Pad で調整できるようにする。 | 完了 |
| FR-06 | 配布可能な Release ビルドの VST3 バンドルを zip として提供する。 | 完了 |
| FR-07 | プラグインと同等機能を持つスタンドアロン実行ファイルを提供する。 | 完了 |
| FR-08 | ビューモード、カメラプリセット、ズーム、色重みなどの UI 状態をセッション間で保持する。 | 完了 |
| FR-09 | リポジトリ内にビルド手順・使用方法・リリース手順のドキュメントを整備する。 | 完了 |
| FR-10 | clap-juce-extensions を用いて CLAP プラグインを生成し、他の成果物と共に配布する。 | 完了 |
| FR-11 | Windows 相当の成果物として macOS AU / VST3 / Standalone / CLAP を提供する。 | 完了 |
| FR-12 | JUCE が提供するモノラル／ステレオから Dolby Atmos 9.1.6（Atmos / ITU 変種）までのレイアウトに自動対応する。 | 完了 |

## 非機能要件
| ID | 説明 | 状況 |
|----|------|------|
| NFR-01 | Outside ビューのジオメトリは常に軸整合の直交ワイヤーフレームを維持する。 | 完了 |
| NFR-02 | Inside ビューは部屋の原点から描画し、カメラが部屋外へ出ないこと。 | 完了 |
| NFR-03 | ウィンドウ幅 1280x720 でも UI が判読でき、ヘッダーコントロールが整列する。 | 完了 |
| NFR-04 | ヒートマップ描画はリファレンスハードウェア（Ryzen 7 3700X）で 1 フレーム 16 ms 以内に更新する。 | 手動確認が必要 |
| NFR-05 | Release ビルド自動化スクリプトはリファレンスハードウェアで 3 分以内に完了する。 | マニフェスト警告 保留 |
| NFR-06 | ドキュメントはコードベースと同期し、タグ付きリリースごとに見直す。 | 完了 |

## 未完了項目 / フォローアップ
- Radiation Heatmap の列カバレッジ（特定列が欠けるとの報告）を検証する。
- Inside Top の向きを調整し、前壁がビュー下部に位置するようにする。
- カメラスイープを自動化したビジュアル回帰テストを検討する。
- マニフェスト生成を無効のままにするか、ネイティブ x64 ヘルパーへ移行するか検討する。
